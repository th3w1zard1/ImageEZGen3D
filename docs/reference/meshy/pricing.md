# Pricing

Simple and flexible pricing.

---

Meshy API for generation is a pay-before-you-go product. You need to purchase API usage based on your application's need before using Meshy API. You can purchase API usage from the [API settings page](https://www.meshy.ai/settings/api). For volume API pricing and custom contracts, please [contact sales](https://share.hsforms.com/1MiMAkL-dSVejYLqOA4V10grx1uq).

The Meshy API pricing is based on credits. Here's a detailed breakdown of the API pricing:
